# To-Do List Web App

A simple and responsive To-Do List application built using HTML, CSS, and Vanilla JavaScript.

## Features

- Add Tasks
- Mark Tasks as Completed
- Delete Tasks
- Local Storage Support
- Responsive Design
- Instant UI Updates

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
